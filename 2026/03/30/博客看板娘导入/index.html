神临

雷迪斯安德宅特们
今天我将为大家带来超级无敌牛福的极品model3.json看板娘如何进入我们一心一意搭建的hexo博客里(注意是hexo呦) hexo

首先让看板娘成功加载到博客

需要的准备

第一步

我们需要去官方sdk下载Cubism SDK for Web
然后解压随便放到个文件夹里从里面搜live2dcubismcore.js找到了就给它复制下来
然后在我们的博客目录里(不是主题的)找到source创建一个肥美的文件夹就叫live2d吧(名字随便起)然后再整个文件夹core把live2dcubismcore.js放里头

这个live2d的教程中不需要任何live2d的插件不然就会在最后显示出那些插件的默认看板娘很难受

第二步

我们要找到我们放代码的地方 一般来说我放的都是博客根目录下\themes\主题\layout_partial\footer.ejs里 这里能在你html页面生成后显示你加的东西(后半句为个人理解)
反正你放那应该就没啥问题 有点话问问GPT吧

第三步(正文开始)

1、

你在footer里需要引用一些外部的库帮助你生成出看板娘 就像这个 他要放到第一行(一定要访问看看是不是有东西)
然后引用你在Cubism SDK for Web里下载解压完后找到的live2dcubismcore.js文件 用本地路径就好这个就放到第二行
再就是引用这个库

像这样就可以了 顺序要对 不然加载不出来

2、

我建议是把我们下载的或者别人仓库里的model3那个整个的文件(包括动作贴纸moc3等文件)整到我们自己的github上 然后再引用到博客里
然后用的路径最好是raw的路径就像https://raw.githubusercontent.com/你的github仓库路径/模型名字.model3.json (因为model3是模型的主体现文件 它是主人)
还有就是要打开这个model3文件看看里面 在你没做更改前内容肯定是 文件夹/文件 但是我们是引用不是本地所以需要把它们全部换成raw的浏览器访问路径这个你可以放心全复制给ai让它帮你整然后你再给它复制粘贴回来

3、

你还可以在你的模型仓库里整一个叫waifu.js的文件内容也可以找ai随便简单生成一下就可以 这个其实等你全整好了也就没啥用了

然后的话我们的第一步也就是让看板娘加载到博客就完成了
这时候我真得吐槽一下GPT了(仅吐槽可跳过不看)(GPT升级版的不用看)

那个GPT是真气人 可能是因为我用的免费版 毕竟绑不了银行卡(同时没钱)
然后呢我那那时候就天天找它问怎么整这种model3的看板娘 它给我的回复真是天花乱坠都是什么”我给你整个一定能行一键导入成的代码”什么”这个代码肯定行你只要复制粘贴然后就能看见看板娘”啥的 真是坑飞我了 我那时候老信它了 说啥我都满怀期待的说 行 好 ok 然后没有一次成功了 每次给他看报错的时候都说你这个什么路径不对那个引入库的顺序不对我再给你生成一个 我靠我现在想起来感觉我是沙子啊 你们千万不要无脑复制粘贴ai的东西 不然等到它生成对的时候你都自己学会了 特别是在我自己想把我的模型放到本地的时候更是不知道在说什么 最牛福的是它甚至会说它眼花了我一个代码后面有逗号它说没有要不是我又问了一遍都不知道我这块是对的 太猎奇了

然后让看板娘变到能看的大小

这个也是十分的难搞啊 而且这个ai好像跟咱看到的不一样导致全权给ai一般都会失败 但它还是有点用的

第一步

这里我比较推荐deepseek 我在用它写代码的时候我去那叫一个666啊 把全部添加的代码复给它然后说一些要求比如大小要多少px啥的它嗷嗷就是一顿写给你 一下就是近百行的代码

第二步

就是这个ai的眼神跟咱不一样 因为我用的是碧蓝航线的这模型 从Live2D Cubism Viewer上看的时候发现它真的巨大有着整整10000x8000px的画布大小要是在咱那浏览器加800x600px的容器里根本显示不全 只会有一个小角落咱看不见但是问那劳seek说它能看见

第三步

手动调大小和位置 这种时候ai就没什么大用了 我们打开他各种修改后的代码然后问它哪方面是关于画布大小 模型偏移 模型大小比例在哪
一般这在在给你生成的代码里了不然就让它给你生成这种的代码然后慢慢改就可以了 改完确定了这个东西的位置大小后就可以自己再向这个ai提要求加东西了
我的代码也可以做些参考

我能保证的是这个你拿去用了把路径改成你github上的raw路径的话在博客上肯定是没问题的可能前面我有点忘了但是应该没题

注意事项
关于这个ai

一开始的时候它给你生成的引用什么库的链接啥的都记得访问一次访问的到才行ai只是负责嗷嗷给你甩链接 当然要是你发现访问不了了倒是可以问问ai它才会给你换新的
同理所有的链接能去访问就都访问一遍 最浪费时间和最容易出错的就是这些
然后是在博客上调完大小位置后最好找ai给你加上个idle动作的那个东西很好看的

我整成这样也算是看板娘了吧 极限最好只能整成这样了
